Fotorama 0.3

1. Include latest version of jQuery and Fotorama on your web page

<head>
  <script type="text/javascript" src="jquery-1.6.1.min.js"></script>
  <script type="text/javascript" src="jquery.fotorama-0.3.js"></script>
  <link rel="stylesheet" type="text/css" href="jquery.fotorama-0.3.css"/>
</head>

2. Add images

<body>
  <div class="fotorama">
    <img src="img/1.jpg"/>
    <img src="img/2.jpg"/>
    <img src="img/3.jpg"/>
  </div>
</body>

3. Activate Fotorama

<script type="text/javascript">
  $(function() {
    $(".fotorama").fotorama();
  });
</script>


No settings for cropping, easing, autoplay, preload, carousel follow and other noise! No problem with multiply galleries, any desired aspect ratio and size.


Position thumbs as you like

$("#fotorama-сt").fotorama({
  thumbsPosition: "center top"
  // string, default "left top"
});

$("#fotorama-rcv").fotorama({
  thumbsPosition: "right center",
  thumbsVertical: true
  // boolean, default: false
});

$("#fotorama-cb").fotorama({
  thumbsPosition: "center bottom"
});

$("#fotorama-lcv").fotorama({
  thumbsPosition: "left center",
  thumbsVertical: true
});


Ajust transition duration,
hide thumbs on mouseout
add caption through img’s alt,

$("#fotorama-bw").fotorama({
  transitionDuration: 100, // number, default: 500
  hideThumbs: true, // boolean, default: false
  caption: $("#fotorama-bw-caption") // jQuery, default: false
});
